# Changelog

## 1.0.0

- Added horizontal and vertical widget layouts.
- Added saved positions per layout.
- Added Lock Position controls in the tray right-click menu.
- Added Always On Top on/off controls in the tray right-click menu.
- Added settings window version label.
- Added adaptive hardware naming for different PCs.
- Added delayed hardware identity refresh after startup.
- Improved startup by hiding the widget until the renderer is ready.
- Built portable Windows release executable.
